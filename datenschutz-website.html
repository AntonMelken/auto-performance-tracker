4. Server-Logdateien
4. Server log files
Automatisch erfasste Zugriffsdaten
Automatically collected access data

Bei jedem Seitenaufruf verarbeitet GitHub Pages automatisch folgende technische Daten in sogenannten Server-Logs:

With every page request, GitHub Pages automatically processes the following technical data in server logs:

  • IP-Adresse des abrufenden Geräts (IPv4 oder IPv6)
  • Datum und Uhrzeit des Zugriffs
  • Aufgerufene URL (Seitenadresse)
  • HTTP-Methode (GET, POST etc.) und HTTP-Statuscode
  • Übertragene Datenmenge in Byte
  • Referrer-URL (von welcher Seite du kommst – sofern übermittelt)
  • User-Agent-String (Browser-Typ, Betriebssystem, Gerätetyp)
  • IP address of the accessing device (IPv4 or IPv6)
  • Date and time of access
  • Requested URL (page address)
  • HTTP method (GET, POST, etc.) and HTTP status code
  • Amount of data transferred in bytes
  • Referrer URL (the page you came from – if transmitted)
  • User-agent string (browser type, operating system, device type)

Zweck: Auslieferung der Website, Erkennung und Abwehr von Angriffen (Sicherheit), Fehlerdiagnose und Sicherstellung der Systemstabilität.

Purpose: Delivery of the website, detection and defence against attacks (security), error diagnosis, and ensuring system stability.

Rechtsgrundlage: Art. 6 Abs. 1 lit. f DSGVO – berechtigte Interessen des Verantwortlichen (technischer Betrieb, Sicherheit). Diese Verarbeitung ist für den Betrieb einer jeden Website technisch zwingend erforderlich.

Legal basis: Art. 6(1)(f) GDPR – legitimate interests of the controller (technical operation, security). This processing is technically mandatory for operating any website.

Speicherdauer: Gemäß den Angaben von GitHub werden Server-Logs für einen Zeitraum von ca. 90 Tagen aufbewahrt und anschließend automatisch gelöscht. Auf diese Logs haben wir keinen direkten Zugriff; sie werden ausschließlich von GitHub verwaltet.

Retention period: According to GitHub's documentation, server logs are retained for approximately 90 days and then automatically deleted. We have no direct access to these logs; they are managed exclusively by GitHub.

Hinweis: Note: Die IP-Adresse wird dabei ausschließlich für die technische Kommunikation (Antwort an deinen Browser) und die Sicherheitsprotokollierung durch GitHub genutzt. Wir selbst führen keinerlei IP-basierte Analyse oder Profilbildung durch. The IP address is used solely for technical communication (responding to your browser) and GitHub's security logging. We ourselves carry out no IP-based analysis or profiling whatsoever.
